Source Highlight

Plugin Identifier: 20000

Summary

Convert source code from plain text to HTML. Supports major programming languages inculding 4D v14 (English and French).

source-highlight

CODE HIGHLIGHT

CODE HIGHLIGHT (inCode; outCode; inLang; outLang)

inCode (TEXT)

The source code in plain text. Line delimiter should be LF.

outCode (TEXT)

The code converted to HTML.

inLang (TEXT)

Input language specifier. Default is "4d".

See content of Resources/source-highlight for list of supported languages.

You can also add/edit these files to extent the library.

outLang (TEXT)

Output language specifier. Default is "html".

Discussion

Convert source code from plain text to HTML.

preferences

CODE GET LANGUAGE LIST

CODE GET LANGUAGE LIST (languages)

languages (ARRAY)

Discussion

Returns the list of supported languages.

CODE Get language folder

path:=CODE Get language folder

path (TEXT)

Discussion

Returns the languges folder. Default is Resources/source-highlight inside the plugin.

CODE Set language folder

success:=CODE Set language folder (path)

path (TEXT)

success (LONGINT)

1 on success. If the folder does not contain any language files, revert to default and returns 0.

On Windows, the API is ANSI not Unicode i.e. avoid characters not supported in the current locale.

Discussion

Change the languges folder. Default is Resources/source-highlight inside the plugin.
